# 2026 年 AISZ-23 操作系统复习资料

本目录收录操作系统课程的复习总览、速查表、知识清单和章节测试；配套作业与答案存放在课程仓库的 `assignments/` 目录。

## 内容说明

- [复习资料总览](index.html)：统一进入速查表、九章知识清单、章节测试和配套作业
- [操作系统复习速查表](操作系统复习速查表.html)：核心概念、算法、公式与常见题型
- `Knowledge List/`：第 01–09 章知识清单
- `Chapter Test/`：第 01–09 章交互式章节测试
- [配套作业与答案](../../assignments/2026_AISZ-23-CS-Reviews/README.md)

## 使用方式

1. 下载本目录以及配套的 `assignments/2026_AISZ-23-CS-Reviews/` 目录，并保持仓库内的相对位置不变。
2. 用浏览器打开 `index.html`，通过总览进入各类资料。
3. 章节测试可直接在浏览器中作答；总览中的作业入口会跳转至 `assignments/` 下的题目和答案。
4. 页面无需安装依赖；部分公式使用 MathJax 渲染，首次加载时需要联网。
